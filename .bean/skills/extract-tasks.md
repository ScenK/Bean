# Extract Tasks

Scan the provided content and extract the actionable items:

- One checkbox line per task, phrased as an imperative ("Email Sam the invoice").
- Include an owner and due date when the text implies one.
- Drop vague aspirations; keep only things someone could actually do.

If there are no real action items, say so instead of padding the list.
